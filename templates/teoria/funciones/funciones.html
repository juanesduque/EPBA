Funciones


¿Qué son?

Las funciones son bloques o segmentos de código que reciben un nombre y los cuales realizan o ejecutan una tarea específica dentro de un programa, esta herramienta es muy útil y utilizada dentro del paradigma de la programación estructurada, ya que permite dividir un programa de gran complejidad en segmentos de código más pequeños donde cada uno cumple una tarea diferente y este se puede repetir una cantidad N de veces dentro de un programa dependiendo de cual sea la necesidad. Dichas funciones tienen algunas reglas a tener en cuenta como parámetros de entrada, su salida, nombre y demás cosas. Además, se sabe que existen varios tipos de funciones las cuales tienen una utilidad diferente y es imporante conocer cada una de ellas con el fin de saber en qué momento y para qué usarlas.


Sintaxis

Para declarar una función se deben tener en cuenta tres cosas. Primero se debe poner el tipo de dato el cual va a almacenar la variable, luego se debe poner el nombre que va a tener dicha variable, este nombre comunmente coinside con el dato el cual va a almacenar esta, pues de esta manera se facilita su uso; por último, aunque no se hace necesario, se le puede asignar un valor inicial a la variable, solo debe poner el signo de igual y posteriormente darle el valor que se desee, finalizando con un punto y coma. 

            
tipo_de_función nombre_función(parámetros de entrada){
    Instrucciones;
}

Tipos de funciones

Dentro de los tipos de funciones que existen en Arduino para las variables se encuentran los números, las letras, las cadenas de texto, los booleanos y entre otro. Dentro de los números se encuentran los enteros, los flotantes y binarios. A continuación se explica un poco de cada tipo para tener en cuenta cuál es el que se desea y se necesita utilizar para cada uno de los programas, además, se mostrará su sintaxis dentro del lenguaje.


Enteros:

Los números enteros son aquellos que contienen a todos los números que hay en la recta real antes y después del cero, sin tener en cuenta los infinitos puntos que hay entre un número y el otro, además, son aquellos que no poseen decimales o fracción, es decir, son exactos. Para declarar una variable con este tipo de dato, se debe poner la palabra "int" seguida del nombre de la variable. 

            
int nombre_variable = valor;

Flotantes o doubles:

Los números flotantes son aquellos que contienen los infinitos números que hay en la recta real antes y después del cero, además, son aquellos que poseen decimales o fracción, es decir, no son exactos y tienen números después del punto o de la coma. Para declarar una variable con este tipo de dato, se debe poner la palabra "float" seguida del nombre de la variable. Por otra parte, se encuentra los doubles que se definen teóricamente igual que el float, su diferencia es que este tipo de dato tiene más bits de almacenamiento, por lo que pueden ser muchos más los número que pueden guardarse después de la coma o el punto. Para declarar o nombrar este tipo de dato se debe poner la palabra "float" 

            
float nombre_variable = valor;
            
            
            
double nombre_variable = valor;

Booleanos:

Los números booleanos son aquellos que contienen valores binarios, es decir, solo tienen dos posibilidades falso o verdades, alto o bajo o, 0 o 1. Estos valores usualmente se obtienen como resultado de expresiones lógicas que al ser evaluada, se obtiene como resultado alguno de estos valores. Para declarar un dato de este tipo se debe poner la palabra "boolt" seguida del nombre de la variable. 

            
bool nombre_variable = valor;

Letras:

Las letras son un tipo de dato en el que sea puede almacenar, como su nombre lo indica, cualquier letra del alfabeto, sin importar si es mayúscula o minúscula, pero solo se puede almacenar una sola letra a la vez. Para declarar una variable con este tipo de dato, se debe poner la palabra "float" seguida del nombre de la variable. 

            
char nombre_variable = "valor";

Para este problema, existe una variación en la cual se pueden guardar letras o frases completas y para esto se declara como vector. Para declarar un vector con este tipo de dato, se debe poner la palabra "float" seguida del nombre del vector. 

            
char[tamaño] nombre_vector = "valor";

Cadenas de texto:

Los string son tipos de datos que permiten almacernar palabras que no contengan caracteres especiales. Para declarar una variable con este tipo de dato, se debe poner la palabra "float" seguida del nombre de la variable. 

            
string nombre_variable = valor;
